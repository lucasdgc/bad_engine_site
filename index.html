Bad Engine 3D

A simple and deliberately limited 3D engine. Written in Go.

Limitation as a decision

Bad Engine is designed to be simple and easy to use. It is not meant to be a full-fledged game engine, but rather a nice tool for learning and experimentation.

For those who want to code their own 3D games, Bad Engine provides a framework to build upon.

You can go straight to code your games instead of browsing convoluted documentation and watching outdated lessons.

Features

Simple and compatible 3D rendering using OpenGL

OpenGL ensures compatibility with a wide range of hardware and platforms, specially older pcs. So no raytracing, no AI upscaling, no global ilumination. Just color textures and basic lighting.

Lightweight and fast

Bad Engine is designed to be lightweight and fast, making it ideal for low-end hardware. A compressed executable is around 1.5MB.

Cross-platform

Bad Engine can target Windows, Linux, and MacOS. It will be ported to Android and WebGL in the future.

Open Source

Bad Engine will soon be released as Open Source.
